INSTRUCTIONS:

When game is imported and started, press N to start a game, then a series of numbers to provide a seed. Game will accept up to Java's long limit.
Press S to start.
Arrow keys handle movement.
N again to start on a fresh world, followed by the new game procedure.
:, then Q, to save and quit the game.
Upon saving, game will exit; on reopening, press L to load the game or R to replay the previous save.
